# ECON1626 Labour Policy Proposal

## Project overview

This repository contains my Assessment 2 policy proposal for ECON1626. The proposal advises the Deputy Secretary, Employment and Skills, Department of Employment and Workplace Relations on how Australia should respond to the labour-market impacts of generative AI.

The brief focuses on clerical, administrative and entry-level knowledge work, where GenAI may affect tasks, skill demand, hiring pathways and workforce adjustment.

## Repository structure

- `proposal.md` – the final policy proposal, including the executive summary, labour-market analysis, policy options, recommended implementation plan, risks and references.
- `README.md` – this file, which explains the purpose of the repository and how to navigate it.

## How to navigate this repository

Open `proposal.md` to read the full policy proposal.

The proposal is organised into the following sections:

1. Executive Summary
2. Problem Definition and Context
3. Analysis of Impacts
4. Policy Options
5. Recommended Package and Implementation Plan
6. Risks & Mitigations
7. References
